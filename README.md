# 📮 Stampzy: The Digital Curated Stamp Collection

[![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)](https://flutter.dev)
[![Architecture](https://img.shields.io/badge/Architecture-Clean-green?style=for-the-badge)](https://pub.dev)
[![Version](https://img.shields.io/badge/Version-v1.0.0--beta-orange?style=for-the-badge)](https://github.com/RakiticVo/my_stamp_collection)

**Stampzy** is a premium mobile experience designed for the modern philatelist. It transforms physical mail and historical specimens into a vibrant, searchable digital vault, preserving the beauty of every notch and perforation.

---

## ✨ Key Features

### 🛠️ Advanced Stamp Sculptor
Craft the perfect digital specimen with our industry-leading edge simulation:
- **Perforation presets**: Legacy Round, Heritage Zigzag, Modern Square, and Fine Dotted.
- **Geometry choices**: Rectangle, Circle, Hexagon, Diamond, Octagon, and Heraldic Shield.
- **Solid border mode**: For a clean, contemporary studio look.

### 🎨 Creative Workbench
- **Smart Stickers**: Add 'Verified', 'Historic', or custom text overlays with tailored color palettes.
- **Buttery-smooth Dragging**: Optimized local-state interaction engine (zero-lag).
- **Studio Filters**: Vintage, Sepia, Vivid, and Warm filters to restore or enhance your specimens.

### 🗃️ Curator's Ledger
- **Smart Geotagging**: Automatically captures location metadata via GPS.
- **Categorization**: Organize by "Nature", "Architecture", "Heritage", and more.
- **SQLite Persistence**: Your collection remains local, private, and lightning-fast.

---

## 🚀 Technology Stack

- **Core**: Flutter & Dart
- **State Management**: BLoC / Cubit (Predictable State Container)
- **Dependency Injection**: GetIt
- **Persistence**: SQLite (via sqflite)
- **Design System**: Custom Vanila CSS-inspired Material 3 with HSL palettes.
- **Location Services**: Geolocator & Geocoding.

## 🛠️ Getting Started

### Prerequisites
- Flutter SDK (Channel Stable)
- Android Studio / VS Code
- A passion for collection!

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/RakiticVo/my_stamp_collection.git
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
