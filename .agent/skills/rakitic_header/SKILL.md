# Rakitic Header Template Skill

This skill provides standardized templates for new file headers to maintain consistent project meta-data.

## Header Format

Every new file MUST include the following block at the very top:

```dart
/// --------------------------------------------------
/// 🔥 Feature: <Feature Name>
/// 👤 Author: Rakitic
/// 📅 Date: <YYYY-MM-DD>
/// 📝 Description:
///   <Description of the file's purpose>
/// --------------------------------------------------
```

## Templates by File Type

### 1. Cubit
```dart
/// --------------------------------------------------
/// 🔥 Feature: <Feature Name>
/// 👤 Author: Rakitic
/// 📅 Date: <YYYY-MM-DD>
/// 📝 Description:
///   Business logic and state management for <Page/Widget>.
/// --------------------------------------------------
```
